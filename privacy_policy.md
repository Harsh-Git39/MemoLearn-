# MemoLearn Privacy Policy

Last Updated: November 14, 2025

## Overview

MemoLearn helps you save and retrieve answers to your search queries. 
We take your privacy seriously.

## Two Storage Options

### 1. Local Storage (Default)

**What it means:**
- Your data stays on YOUR device only
- Nothing sent to our servers
- Complete privacy
- Works offline

**What we collect:**
- NOTHING. All data stays local.

**Limitations:**
- No sync across devices
- Limited to Chrome storage quota (10MB)
- Basic query matching only

### 2. Cloud Storage (Optional - Requires Your Consent)

**What it means:**
- Your data stored on our secure server
- Enables advanced features
- Allows sync across devices
- Better query matching with AI

**What we collect (only if you opt-in):**
- Search queries you explicitly save
- Text selections you pin as answers  
- Timestamps of saves
- Anonymous usage statistics

**What we DON'T collect:**
- Browsing history (except queries you save)
- Personal information (name, email, etc.)
- Passwords or credentials
- Sensitive data (filtered automatically)

## How We Use Your Data

If you enable cloud storage:

✅ **We DO:**
- Store data encrypted (HTTPS in transit)
- Use it only for extension features
- Keep it private to you only
- Delete it when you request

❌ **We DON'T:**
- Sell your data
- Share with third parties
- Use for advertising
- Track you across websites
- Read data you don't explicitly save

## Data Security

**Technical Measures:**
- HTTPS encryption for all transfers
- Secure server (Render.com, USA)
- No public access to database
- Regular security updates

**Human Measures:**
- Only developer has server access
- No data mining or analysis
- No employees can view your data

## Your Rights

**You can:**
1. View all your data (click extension icon)
2. Delete individual queries (click delete button)
3. Delete ALL data (email support@memolearn.com)
4. Export your data (email support@memolearn.com)
5. Switch to local-only mode anytime
6. Revoke consent anytime

**Response time:** Within 48 hours
